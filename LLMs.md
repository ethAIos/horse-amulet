# LLM Resources

Use these resources when researching or implementing this mod. Prefer official, versioned documentation first.

- NeoForge docs: https://docs.neoforged.net/docs/1.21.1/  
  Primary reference for NeoForge APIs, registries, items, events, resources, recipes, datagen, and testing.

- NeoForge getting started: https://docs.neoforged.net/docs/1.21.1/gettingstarted/  
  Use for MDK workflow, Gradle commands, run configs, Java 21 requirements, and basic project setup.

- ModDevGradle docs: https://docs.neoforged.net/toolchain/docs/plugins/mdg/  
  Use for `build.gradle`, run configurations, data generation, and Gradle plugin behavior.

- NeoForge data components: https://docs.neoforged.net/docs/1.21.1/items/datacomponents/  
  Use when storing filled amulet data on `ItemStack`s.

- NeoForge recipes: https://docs.neoforged.net/docs/1.21.1/resources/server/recipes/  
  Use for the string/leather amulet recipe and optional datagen.

- Minecraft Wiki entity format: https://minecraft.wiki/w/Entity_format  
  Secondary reference for horse entity data and persisted NBT fields.

- Minecraft modding tutorials, such as Kaupenjoe's NeoForge 1.21.1 material (https://www.youtube.com/playlist?list=PLKGarocXCE1G6CQOoiYdMVx-E1d9F_itF), or CurseForge Academy (https://www.youtube.com/watch?v=CXuG9Mfhmwo).
  Use only as a learning aid; confirm final APIs against NeoForge docs and local MDK code.
